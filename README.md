# qeom

art via [quil](https://github.com/quil/quil) inspired by [geometrydaily](http://geometrydaily.tumblr.com/) and [PATAKK](http://patakk.tumblr.com/).

### Usage

lein run <i>number</i>

0 &lt;= <i>number</i> &lt;= 7
